Чистая установка

Где скачать

Официальная страница Amnezia дает актуальные версии для iOS, Android, Windows, macOS и Linux. Сначала используйте ее, а затем уже магазин приложений на устройстве.

Скачать Amnezia
iOS

iPhone и iPad

  1. Откройте официальную страницу установки и перейдите к варианту для iOS.
  2. Установите Amnezia и откройте приложение.
  3. Скопируйте личный ключ, который выдал администратор.
  4. В Amnezia нажмите добавление профиля и выберите импорт из буфера.
  5. Разрешите системное добавление профиля, если iOS попросит подтверждение.
  6. Включите профиль Noctalia. Статус должен стать активным.
Android

Телефон или планшет

  1. Откройте официальную страницу установки и выберите Android.
  2. Установите Amnezia из магазина приложений или другим официальным способом.
  3. Скопируйте личный ключ, который выдал администратор.
  4. В Amnezia нажмите добавление профиля и выберите импорт из буфера.
  5. Подтвердите системный запрос на создание защищенного маршрута.
  6. Если Android ограничивает работу в фоне, разрешите Amnezia работать без экономии батареи.
Windows

Windows 10/11

  1. Скачайте установщик Amnezia для Windows с официальной страницы.
  2. Установите приложение и запустите его.
  3. Скопируйте личный ключ, который выдал администратор.
  4. Добавьте новый профиль через импорт из буфера или ручную вставку.
  5. Включите профиль Noctalia. Если страницы не открываются, запустите Amnezia от имени администратора.
macOS

MacBook и iMac

  1. Скачайте Amnezia для macOS с официальной страницы.
  2. Перетащите приложение в папку Applications.
  3. Если macOS предупреждает о приложении, откройте его через правый клик и пункт Open.
  4. Импортируйте личный ключ из буфера обмена.
  5. Разрешите создание системного профиля в настройках, если macOS попросит подтверждение.
Linux

Ubuntu, Debian, Mint

  1. Скачайте Linux-версию Amnezia с официальной страницы.
  2. Откройте загруженный файл тем способом, который предлагает ваш дистрибутив.
  3. Если система просит разрешение на запуск, подтвердите его только для скачанного файла.
  4. Откройте Amnezia и импортируйте личный ключ.
  5. При запросе прав администратора подтвердите действие, чтобы профиль смог работать на уровне системы.
Общее

Как понять, что все включено

  1. В Amnezia выбран ваш профиль Noctalia.
  2. Статус связи активен.
  3. Страницы открываются без ошибок, а адрес сети отличается от обычного.
  4. Если профиль перестал работать, запросите у администратора новый ключ.

Если не включается

Быстрый чек-лист

  • Проверьте, что ключ скопирован целиком, без лишних пробелов в начале и конце.
  • Обновите Amnezia до последней версии.
  • Проверьте дату и время на устройстве, включите автоматическую синхронизацию.
  • Попробуйте другую сеть: Wi-Fi вместо мобильной сети или наоборот.
  • Удалите старый профиль и импортируйте ключ заново.
  • Не передавайте ключ другим людям: при подозрении на утечку он может быть перевыпущен.
